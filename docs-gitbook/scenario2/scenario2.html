Statement Of Work

Scenario 2 - IoT

Level: Hard

Prerequisites: Scenario 1 completed and working. Basic understanding of messaging concepts related to AMQP/MQTT/IoT.

Outcome: Successfully deployed AMQ Online IoT features which are tech preview and produced/consumed messages via MQTT and AMQP.

SoW

  1. Assist client deploying IoT features onto AMQ Online cluster.

  2. Assist client configuring a AMQ Online cluster to support IoT.

  3. Assist client in testing the cluster is working correct by producing and consuming messages via AMQP/MQTT.
